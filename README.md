# cybersecurity-home-labs
Hands-on cybersecurity and networking labs with VirtualBox, pfSense,  and Windows environments
This repository documents hands-on networking and cybersecurity labs built using pfSense, VirtualBox, and Windows systems.

The goal is to simulate enterprise-style network segmentation, firewalling, and troubleshooting while documenting real-world issues encountered during implementation.

## Tools
- pfSense CE
- VirtualBox
- Windows Server
- Windows 11
- Kali Linux

## Focus Areas
- Network segmentation
- Firewall rule design
- DNS and routing behavior
- Troubleshooting methodology
